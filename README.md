Multi-Threaded Banking System - Operating Systems Project

This project demonstrates concurrency issues using pthreads.

Phase 1 - Race condition demonstration
Phase 2 - Mutex protection
Phase 3 - Deadlock demonstration
Phase 4 - Deadlock prevention

Build Instructions:
gcc -pthread phase1.c -o phase1
gcc -pthread phase2.c -o phase2
gcc -pthread phase3.c -o phase3
gcc -pthread phase4.c -o phase4
